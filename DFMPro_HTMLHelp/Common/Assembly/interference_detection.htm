DFMPro for Creo Parametric

Interference Detection "干渉検出"

ルール

目的とする機能と組み立てやすさを実現するために、アセンブリ構成部品間の干渉を避けてください。


説明

このルールは、アセンブリ構成部品間の干渉状態を特定します。干渉検出は複雑なアセンブリで特に有用であり、構成部品同士が干渉しているかどうかを可視化して判断するのが難しい場合に役立ちます。


推奨事項

デフォルトでは、すべてのアセンブリ構成部品が解析対象として選択され、すべてのキルト（Quilt）サーフェスは干渉検出から除外されます。一般に、構成部品の公称寸法（ノミナル寸法）により干渉が発生しないことが期待されます。


構成

干渉検出ルールは、次の2つの方法で構成できます：

  1. すべての構成部品をチェック

    これはデフォルトで選択されています。アセンブリ内のすべての構成部品の組み合わせ（ペア）について、干渉状態がチェックされます。

  1. 選択した構成部品をチェック

    デフォルトではチェックされていません。アセンブリ内で選択した構成部品ペア間の干渉状態がチェックされます。


列（Component 1 と Component 2）の下に構成部品を追加する：

ルール構成で構成部品を追加する方法は3つあります：

  1. 列に構成部品名を手動で入力します。

  2. 右クリックしてファイルの場所を参照（browse）します。この手順を Component 1 と Component 2 の両方に対して実行します。

  3. キーボードで Ctrl+I を押して、MS Excel ファイルをインポートします。この Excel ファイルには、各列に追加された構成部品名が入った2つの列が含まれます。Excel ファイルをインポートすると、データは自動的に Rule Manager に取り込まれます（下の例を参照）。

標準テンプレート形式

除外（Ignored）構成部品

干渉検出の検証中に、構成部品のリストを除外できます。この条件は、許容される干渉を引き起こすことが分かっている特定の構成部品を含めた解析を行う場合に、ユーザーが望む干渉結果のみを得られるようにするために役立ちます。


Components 列の下に除外構成部品を追加する：

ルール構成で除外構成部品を追加する方法は3つあります：

  1. 構成部品名を手動で入力します。

  2. 右クリックしてファイルの場所を参照し、パーツまたはアセンブリ名を選択します。

  3. キーボードで Ctrl+I を押して、MS Excel ファイルをインポートします。この Excel ファイルには、各行に追加された構成部品名が入った1列が含まれます。Excel ファイルをインポートすると、データは自動的に Rule Manager に取り込まれます（下の例を参照）。

以下のパーツタイプ間の干渉がチェックされます：

  1. ソリッド同士（Solid to Solid）

  2. ソリッドとキルト（Solid to Quilt）

  3. パーツ（ソリッドまたはキルト）とサブアセンブリ（Part (Solid or Quilt) to sub-assembly）

  4. サブアセンブリ同士（Sub-assembly to sub-assembly）

  5. ハイブリッドパーツ（ソリッドとキルトの両方を含む）（Hybrid Parts (having solids as well as quilts)）


追加オプション


Option

Function

Ignore Interference Volume

構成された値より小さい干渉体積（Interference volume）は無視されます。

Ignore Quilt Surfaces

アセンブリファイル内のすべての構成部品／キルトサーフェスが無視されます。

Ignore Threaded Region

ルール検証中、ねじ穴と留め具の間の実際の干渉領域（インターフェース）が無視されます。
