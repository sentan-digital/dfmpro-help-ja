DFMPro for Creo Parametric

ダイカスト部品チュートリアル


ダイカスト部品のワークフローは、角柱ミル加工部品（Prismatic Milled Part）と同じです。

注記：


ダイカスト部品の解析を実行する前に、DFMPro ユーザーインターフェースの［Inputs］グループボックスで、製造プロセス（Manufacturing Process）オプションを「Die Casting」に設定してください。

ダイカストのサンプル部品は、DFMPro のインストールフォルダ内の Examples\Die Casting フォルダに用意されています。

Analysis ボタンをクリックすると、入力選択（Input Selection）ダイアログボックスが表示されます。引き抜き方向を定義するために、平面または平坦な基準面を選択してください。選択内容は変更することができ、基準平面の選択方向を反転させることも可能です。同じダイアログボックス内には、Nominal Part thickness（公称部品肉厚）の計算に関する入力項目もあります。公称部品肉厚をソフトウェアで自動計算する場合は Compute Automatically オプションを選択し、手動で設定する場合は Set Value オプションを選択して、入力ボックスに肉厚値を入力してください。

DFMPro に標準で提供されているダイカストルールは、アセンブリに対しては実行されません。ダイカストルールは、個々の部品に対して実行してください。
