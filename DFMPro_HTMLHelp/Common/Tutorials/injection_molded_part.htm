DFMPro for Creo Parametric

射出成形パーツ チュートリアル

射出成形パーツのワークフローは、プリズマティック加工パーツの場合と同じです。


注記：

  • 射出成形パーツの解析を実行する前に、DFMPro ユーザーインターフェースの Inputs グループボックスで、Manufacturing Process オプションを Injection Molding に設定してください。

  • 射出成形のサンプルパーツは、DFMPro のインストールフォルダ内の Examples\Injection Molding フォルダに用意されています。

  • Analysis ボタンをクリックすると、Input Selection ダイアログボックスが表示されます。引き抜き方向を定義するために、平面または平坦な参照面を選択してください。選択オプションは変更することができ、参照平面の選択方向を反転することも可能です。同じダイアログボックス内には、Nominal Part thickness（公称パーツ肉厚）の計算に関する必須入力項目もあります。公称パーツ肉厚をソフトウェアで自動計算する場合は Compute Automatically オプションを選択し、値を指定する場合は Set Value オプションを選択して Input ボックスに肉厚の値を入力してください。

  • DFMPro に標準で用意されている射出成形ルールは、アセンブリに対しては実行されません。射出成形ルールは、個々のパーツに対して実行してください。
