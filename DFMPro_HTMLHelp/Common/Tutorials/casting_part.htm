鋳造部品チュートリアル

鋳造部品のワークフローは、プリズマティック切削部品の場合と同じです。


注記：

  • 鋳造部品の解析を実行する前に、DFMPro ユーザーインターフェースの［Inputs］グループボックスで、製造プロセス（Manufacturing Process）オプションを「Casting」に設定してください。

  • 鋳造のサンプル部品は、DFMPro のインストールフォルダ内にある Examples\Casting フォルダに用意されています。

  • Analysis ボタンをクリックすると、入力選択（Input Selection）ダイアログボックスが表示されます。引き抜き方向を定義するために、平面または平坦な参照面を選択してください。選択オプションは変更可能で、参照平面の選択方向を反転させることもできます。同じダイアログボックス内には、Nominal Part thickness（公称部品肉厚）の計算に関する入力項目もあります。公称部品肉厚をソフトウェアで自動計算する場合は Compute Automatically オプションを選択し、値を手動で指定する場合は Set Value オプションを選択して、入力ボックスに肉厚の値を入力してください。

  • DFMPro に標準で用意されている鋳造ルール（Standard Casting Rules）は、アセンブリに対しては実行されません。鋳造ルールは、個々の部品に対して実行してください。
