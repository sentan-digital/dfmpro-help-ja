DFMPro for Creo Parametric

エンティティ選択


  1. User Inputs グループボックスで Analyze ボタンをクリックすると、Entity Selection グループボックスが表示されます。

  2. 主抜き方向はデフォルトで選択されており、モデルはデフォルトの金型（コア／キャビティ／その他）の面分類に基づいて解析されます。

  3. DFMPro では、一次および二次の抜き方向を定義するために、グラフィックス領域から Plane または Planar face または Part Edge または Axis または Non-planar faces（円すい面および円筒面） を選択できます。

  4. 主抜き方向については、定義済みの抜き方向を変更するために Select Entity ボタンをクリックします。選択内容は、以前に選択されていた面を置き換え、Entity Selection グループボックスに新しい抜き方向 ID が表示されます。

  5. 二次抜き方向については、部品の複雑さに応じて、アンダーカット／サイドアクション用としてメーカーに複数の二次抜き方向を提示できます。

  6. Select Entity ボタンをクリックし、必要な抜き方向を選択します。同じ手順を繰り返して、フィールドに複数の必要な抜き方向を追加します。

選択された二次抜き方向 ID は、Entity Selection グループボックスに表示されます。

機械加工セットアップ


DFMPro は、Machining Setup を表示および編集するためのオプションを提供します。セットアップ情報に基づいて、設計者は設計のフィーチャーを最適化できます。
