DFMPro for Creo Parametric

DFMPro の使い方

DFMPro は、設計者向けのソフトウェアであり、製造性を上流工程で検証し、製造が困難・高コスト、または製造不可能な設計領域を特定することを支援します。

DFMPro は Creo Parametric 内で動作するアドインアプリケーションです。DFMPro for Creo Parametric をインストールすると、protk.dat ファイルを使用して、DFMPro セットアッププログラムが Creo Parametric の補助アプリケーションとして自動的に有効化されます。


DFMPro を起動して設計解析を実行する

  1. Creo Parametric を起動します。

  2. 単一のソリッドボディを含む パーツ／アセンブリ ドキュメントを開きます。

  3. DFMPro メニューをクリックします。

  4. DFMPro メニュー内の Start DFMPro をクリックして、ユーザーインターフェースを起動します。

  5. ルール構成用のデフォルトのルールファイルが、DFMPro ユーザーインターフェースに表示されます。このファイルでは、すべての設計に対してすべてのルールが有効になっています。


注記:

  • 既存のルールファイルを使用するか、組織の設計環境において 追加のルールファイルを作成 してください。


  1. ドロップダウンリストから適切な 製造プロセス を選択します：積層造形、ケーブリング、鋳造、ダイカスト、基本設計ルール、射出成形、精密鋳造、熱成形、フライス加工、砂型鋳造、板金加工、板金成形、チューブ配管、真空樹脂注入成形、溶接、旋盤加工、または組立て。

  2. User Inputs グループボックスで、必要な入力パラメータを定義します。

  3. DFMPro メニューでRule Managerボタンをクリックし、ルールマネージャーダイアログボックスを開きます。

  4. 選択した製造プロセスに応じて実行するために必要な ルールのチェックボックス をリストから選択します。


注記:

  • ユーザーは事前定義されたルールファイルを使用することも、Configure Rule ボタンをクリックしてデフォルト値を変更することもできます。また、設計環境内で追加のルールファイルを作成することも可能です。


  1. 解析を実行して、ルールに対してパーツを検証します。

  2. 実行後、DFMPro では 結果を表示して解釈 することができます。

  3. ユーザーは エラーを修正して再検証 することができます。


注記:

  • 報告されたエラーが重要でない場合は、今後の解析実行において 不要なエラーとして指定 できます。


  1. 設計を レビュー担当者 に提出して検証してもらいます。

DFMPro によってすでに検証が行われているため、レビュー担当者は同じ検証手順を繰り返す必要はありません。DFMPro が解析できなかった領域および報告された箇所のみを確認すればよく、手動レビューではそれらの領域に重点を置いてください。

通常、レビュー担当者は DFMPro で自動化されていない設計上の観点について確認を行います。


DFMPro の有効化／無効化

必要に応じて手動で有効化・無効化する手順：

  1. Tools をクリックして補助アプリケーションを開きます。

  2. 補助アプリケーションで DFMPro オプションを選択します。

  3. Start ボタンをクリックして DFMPro を起動します。

  4. Start ボタンをクリックしてアプリケーションを追加します。

  5. DFMPro を無効化するには、Stop ボタンをクリックします。

  6. アプリケーションを完全に削除するには、Delete ボタンをクリックします。

  7. Close ボタンをクリックします。


DFMPro の使い方を素早く習得するには、Prismatic Milled Part チュートリアルの演習を参照してください。

この例はフライス加工を対象としていますが、ワークフローは、積層造形、ケーブリング、鋳造、ダイカスト、射出成形、精密鋳造、砂型鋳造、板金加工、板金成形、熱成形、チューブ配管、旋盤加工、真空樹脂注入成形、組立て、および溶接アセンブリでも同じです。
