DFMPro for Creo Parametric

Extruded Hole Distance to Cutouts "押出穴とカットアウト間の距離"

ルール

押し出し穴と切り欠きとの最小距離を板厚で割った比率は、ユーザーが指定した値以上である必要があります。


説明

押し出し穴から切り欠きまでの距離が最小値を下回ると、対象領域において板材の変形や割れが発生する可能性があります。押し出し穴がエッジに近すぎる場合、エッジが変形して膨らみが生じることがあります。


推奨事項

一般的なガイドラインとして、押し出し穴と切り欠きとの最小距離は、板厚の少なくとも4倍以上とする必要があります。


構成

DFMPro は押し出し穴フィーチャーを識別し、押し出し穴と切り欠きとの最小距離を板厚で割った比率を算出します。この実際の比率は、設定された比率と照合して検証されます。

ユーザーは、デフォルトで設定されている比率を編集することができます。

ルール UI では、材料の一覧が マップマテリアル から読み込まれ、ユーザーは材料およびその値を設定することができます。

材料とその値を設定した後、ユーザーは Ctrl+M コマンドを使用して、ルール UI 上で材料グループおよびグレードごとに設定済み材料の一覧をフィルタ表示できます。同じキー（Ctrl+M）を使用してフィルタを解除することもできます。
