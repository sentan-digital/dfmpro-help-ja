Tear Drop Hem "ティアドロップヘム"

ルール

  • ティアドロップヘムの半径と板厚の比は、設定可能な指定値以上である必要があります。デフォルトの比率は 0.5 です。

  • リターンフランジ高さと板厚の比は、設定可能な指定値以上である必要があります。デフォルトの比率は 4.0 です。

  • 開口部（ヘム端と部品との間隔）と板厚の比は、設定可能な指定値以上である必要があります。デフォルトの比率は 0.25 です。


構成

ルール UI では、材料の一覧は Map Material から読み込まれ、ユーザーは材料およびそれぞれの値を設定できます。

材料とその値を設定した後、ユーザーは Ctrl+M コマンドを使用して、ルール UI 内で Material Group とグレードを含む設定済み材料の一覧をフィルタリングして表示できます。同じキー（Ctrl+M）を使用してフィルターをリセットできます。
