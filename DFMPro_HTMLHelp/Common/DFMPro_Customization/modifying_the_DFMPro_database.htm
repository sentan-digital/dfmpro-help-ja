DFMPro データベースの変更

DFMPro には、標準穴径（Standard Hole Sizes）を含むデータベースが用意されています。データベース内のデフォルト値は、データベースで定義されている材料に基づいて利用可能です。DFMPro のデータベースは、すべてのルールファイルから読み取られます。

このデータベースは、使用している加工環境で用いられる材料および穴径を反映するように変更することができます。


  1. DFMPro メニューの Rule Manager ボタンをクリックして、Rule Manager ダイアログボックスを開きます。


  2. Standard Hole Sizes ルールを選択し、Edit Rule ボタンをクリックします。

    現在のレコード一覧が表示されます。各レコードは、標準ドリルビット（すなわちドリル径）を定義しています。


  3. 必要に応じてサイズを Add（追加）または remove（削除）し、その後 OK ボタンをクリックします。


  4. Save ボタンをクリックします。

    これらの変更はデフォルトのデータベースファイルに保存され、すべてのルールファイルから読み取られます。


注記:

  • DFMPro は、オープンソースデータベースである SQLite を使用して、材料および標準穴径のデフォルトデータベースを提供しています。このデータベースは *.dat ファイルとして保存されており、上記で説明した方法で変更することができます。
