C++ を使用したルールのプログラミング


DFMPro では、C++ プログラミングを使用して新しいルールを定義する方法が提供されています。

新しいルールを定義するには


  1. 開発環境（例：Microsoft Visual Studio）を起動します。この環境では、コードの記述、コンパイル、およびバイナリ（DLL）としてのリンクが可能です。

この開発環境では、

  • ルール定義時に、複雑なロジックやコンパイル処理を使用できます。

  • エラーエンティティのハイライト表示を行うために、表示オプションをオーバーライドできます。

  • ルールおよびそのパラメータを構成するためのインターフェースを作成できます。

  1. ルールのプログラミングが完了したら、DFMPro のルールマネージャーに戻り、ルールモジュールの登録 機能を使用して、ルールのバイナリを登録します。

注:

この方法の欠点は、ルールロジックの編集や表示が容易ではない点です。編集または表示を行うには、再度開発環境に切り替えて、上記の手順を実行する必要があります。

API（Application Programming Interface）の詳細については、テクニカルサポートまでお問い合わせください。
