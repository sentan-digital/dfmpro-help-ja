Minimum Radius at Base of Boss "ボス根元部の最小半径"

ルール

ボス根元部の最小半径は、ユーザーが指定した値に従う必要があります。


説明

ボスは、多くの部品設計において、部品同士を取り付けるための取付点や、締結部品を組み付けるために使用されます。最も一般的な形状は、ねじ、ねじ込みインサート、またはその他の締結用ハードウェアを受け入れるための穴を備えた円筒状の突起です。

ダイカスト工程では、凝固中に合金がコアに沿って収縮し、その結果、周囲全体に分散されるべき応力が、鋭角なコーナー部に集中してしまいます。これにより部品が弱くなり、場合によっては破断に至ることもあります。また、この部分は局所的な熱および応力の集中点となり、最終的には割れや早期破損を引き起こす可能性があります。

応力を低減するために、ボスの根元部には所定の最小半径を持つ適切なフィレットを設ける必要があります。ただし、フィレット半径が大きすぎると、ボス根元部に肉厚部が形成されてしまうため注意が必要です。


推奨事項

一般的に、ボス根元部の最小半径は、構成で設定された値に従う必要があります。


構成

ルール UI では、材料の一覧が 材料マッピング から読み込まれ、ユーザーは材料およびそれらの値を構成することができます。

材料とその値を構成した後、ユーザーは Ctrl+M コマンドを使用して、ルール UI 上で材料グループおよびグレードとともに、構成済み材料の一覧をフィルタリングして表示できます。同じキー（Ctrl+M）を使用して、フィルタをリセットできます。
