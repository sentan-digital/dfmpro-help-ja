DFMPro for Creo Parametric

Spacing between Cutouts "カットアウト間隔"

ルール

ウェルドラインの発生を防ぐために、カットアウト間には十分な距離を確保する必要があります。


説明

特に「ニットライン」や「ウェルドライン」が発生する可能性がある場合、強度を確保するためにカットアウト間の十分な間隔を維持する必要があります。


推奨事項

一般的なガイドラインとして、カットアウト間の最小推奨距離は、部品の公称肉厚の2倍以上である必要があります。


構成

DFMPro はカットアウト間の距離を識別し、距離と肉厚の比率を算出した上で、設定された比率と照合して検証します。

ユーザーは、デフォルトで構成されている比率を編集することができます。

ルール UI では、材料の一覧が マップマテリアル から読み込まれ、ユーザーは材料およびその値を構成することができます。

材料とその値を構成した後、ユーザーは Ctrl+M コマンドを使用して、ルール UI 内でマテリアルグループおよびグレード別に構成された材料の一覧をフィルタして表示できます。同じキー（Ctrl+M）を使用してフィルタをリセットすることもできます。
