Rib Reinforcement Check "リブ補強チェック"

ルール

リブの高さ、幅、または断面積を部品肉厚で割った比率は、ユーザーが指定した値未満である必要があります。この指針に違反した場合、リブは補強リブによって補強する必要があります。


説明

リブは、プラスチック部品上に設けられる縦方向の突起で、さまざまな目的で使用されます。部品の強度を高めるため、あるいは部品設計に装飾的な要素を加えるために使用されることがあります。

リブの幅や高さが肉厚に対して過度に大きい場合、構造的な健全性に影響を及ぼす可能性があります。そのようなリブが高荷重を受ける場合には、補強リブを追加する必要があります。


推奨事項

一般的な指針として、リブ高さはさまざまな検討事項に基づき、肉厚の 3 倍以内に制限されます。リブ幅（および断面積）に関する指針は、使用条件に基づいて策定する必要があります。


構成

自立リブの場合、リブ高さは支持面に対する法線方向に測定されます。支持面の法線が変化する場合や、リブが複数方向から支持されている場合には、引き抜き方向が高さ測定の基準として考慮されます。測定されるリブ断面積は近似値となります。

製品の製造条件および使用条件に合わせて、高さ比率および断面積比率の適切な値を設定してください。

ルール UI では、材料の一覧が Map Material から読み込まれ、ユーザーは材料およびその値を設定することができます。

材料およびその値を設定した後、ユーザーは Ctrl+M コマンドを使用して、ルール UI 内で材料グループおよびグレードごとに設定済み材料の一覧をフィルタ表示することができます。同じキー（Ctrl+M）を使用して、フィルタをリセットすることもできます。
