Real-time DFMPro FAQ

Real-time DFMPro には別途ライセンスが必要ですか？

いいえ。Real-time DFMPro は別途ライセンスを必要としません。一次の DFMPro ライセンスが利用可能であり、その中に Real-time DFMPro ライセンスが含まれている場合に実行されます。


Real-time DFMPro はどのバージョンの Creo Parametric をサポートしていますか？

Real-time DFMPro は以下のバージョンをサポートしています：

  • Creo Parametric 12.4

  • Creo Parametric 11.0

  • Creo Parametric 10.0

  • Creo Parametric 9.0

  • Creo Parametric 8.0


Real-time DFMPro がサポートする具体的なモジュールは何ですか？

DFMPro は、以下の製造プロセスに関連するルールをサポートしています：

  • 積層造形

  • 射出成形

  • プリズマティックミル

  • 板金


Real-time DFMPro はアセンブリをサポートしますか？

いいえ。現時点では、Real-time DFMPro はアセンブリをサポートしていません。


Real-time DFMPro は部品全体を解析しますか？

いいえ。Real-time DFMPro は、Real-time を有効にした後に作成／変更されたフィーチャーを解析します。


Real-time DFMPro は Creo Parametric モデルのファミリーテーブルインスタンスをサポートしますか？

はい。Real-time DFMPro はファミリーテーブルインスタンスをサポートします。


Real-time DFMPro はコピー、ミラー、およびブーリアン機能で作成されたフィーチャーをサポートしますか？

いいえ。Real-time DFMPro はコピー、ミラー、およびブーリアン機能をサポートしていません。


Real-time DFMPro は Windchill ワークスペースから開いた部品を解析しますか？

はい。Real-time DFMPro は Windchill ワークスペースから開いた部品を解析します。
