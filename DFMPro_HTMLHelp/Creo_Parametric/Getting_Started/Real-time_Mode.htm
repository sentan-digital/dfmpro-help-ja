Real-time DFMPro の使用方法


Real-time DFMPro は、フィーチャーの作成／修正時に、迅速かつ即時のフィードバックおよび推奨事項を提供します。

Real-time DFMPro のオプションは、DFMPro メニューから利用できます。

Real-time DFMPro を起動して設計解析を実行する：


  1. Creo Parametric を起動します。

  2. 単一のソリッドボディを含むパーツドキュメントを開くか、新規にパーツを作成します。

  3. 以下のいずれかの方法で Real-time DFMPro を開始できます：

  1. 方法 1： 既存のパーツファイルを開く、または新しいパーツファイルを作成すると、デフォルトで Real-time DFMPro インターフェースが表示されます。
  2. 方法 2： Real-time アイコンをクリックして Real-time DFMPro を開始します。

  1. 適切なプロセスを選択します：積層造形（Additive Manufacturing）、プリズマティックミル（Prismatic Mill）、板金（Sheet Metal）、または 射出成形（Injection Molding）。

  1. Select ボタンをクリックします。

プロセスを選択すると、ユーザーがフィーチャーを作成／修正した場合にのみ、Real-time DFMPro が検証を行い、推奨事項を提供します。

  1. 検証後、オレンジ色のアイコンが表示されます。オレンジ色のアイコンをクリックすると、推奨事項を確認できます。

推奨事項表示

  1. クイックヘルプを起動するには、（以下に示す）Arrow アイコンをクリックします。

クイックヘルプ表示

  1. 複数の推奨事項がある場合は、ナビゲーションキーを使用してすべての推奨事項を表示します。

  1. 推奨事項をリスト形式で表示するには、左側のボタンをクリックします（以下参照）。

  1. 設計変更中に一部の DFMPro 推奨事項を無視またはスキップした場合、それらは Ignore リストに保存されます。スキップまたは無視された DFMPro 推奨事項を確認するには、 Ignored Instances ボタンをクリックします。

注：

DFMPro は、現在の Creo Parametric セッションに対して Ignore リストを記憶します。

  1. 緑色のアイコンをクリックすると、DFMPro ルールが合格したことを示すメッセージが表示されます。

DFMPro Rules リンクをクリックすると、合格したルール名の一覧を表示できます。

ルール名をクリックすると、クイックヘルプが起動します。

  1. モデル全体を解析するには、here リンクをクリックして、デフォルトの DFMPro ユーザーインターフェースを起動します。

射出成形


射出成形プロセスを選択した後、 Select ボタンをクリックして、主引き抜き方向を定義します。

Real-time DFMPro はデフォルトの方向を提供しますが、ユーザーは Select Direction ボタンをクリックし、平面上の方向フェースを選択することで引き抜き方向を変更できます。必要に応じて、 Flip Direction ボタンをクリックして方向を反転できます。

Real-time DFMPro では、新しいフィーチャーを追加する場合と既存フィーチャーを修正する場合とで、公称肉厚の計算方法が異なります。

公称肉厚を更新または再計算するには、 Recalculate wall Thickness ボタンをクリックします。

積層造形


積層造形プロセスを選択した後、 Select ボタンをクリックして、造形方向を定義します。


Real-time DFMPro はデフォルトの造形方向を提供しますが、ユーザーは Select Direction ボタンをクリックし、平面上の方向フェースを選択することで造形方向を変更できます。必要に応じて、 Flip Direction ボタンをクリックして方向を反転できます。

Real-time DFMPro を終了する手順


必要に応じて、以下の手順に従って、Real-time 解析プロセスを途中でキャンセルできます：

  1. DFMPro メニューから Real-time を起動します。

  2. Cancel ボタンをクリックします。

注：

いずれの操作（ 選択、または キャンセル）も、現在の Creo パーツのセッションにのみ適用されます。

Real-time DFMPro ルールファイルの挙動


デフォルトでは、ルールファイルで有効化されているサポート対象ルールが検証に使用されます。サポート対象ルールが選択されていない場合、Real-time DFMPro はデフォルトのルール構成値に基づいて推奨事項を提供します。
